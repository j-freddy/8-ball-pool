8 Ball Pool
--------------------------------------------------------------------------------

Inspiration
--------------------------------------------------------------------------------
8 Ball Pool, Miniclip

Gameplay
--------------------------------------------------------------------------------
- 2 players
- 15 balls to pot
	- 7 red* balls, 7 blue* balls
	- 1 black ball
		- this is the 8 ball

*colours can be customisable

Measurements
--------------------------------------------------------------------------------
- pool table: 1:2 ratio
- pool table inc. frame: 122cm x 213cm

Colour palette
--------------------------------------------------------------------------------
495867
577399
bdd5ea

Programming style
--------------------------------------------------------------------------------
- classes and objects
	- declare all properties outside the constructor
	- assign constants outside the constructor
	- assign object references inside the constructor
	- assign all other properties inside the constructor
	- each object has 2 methods that are called each frame/tick while active
		- update
			- handles logic
		- draw
			- handles GUI
